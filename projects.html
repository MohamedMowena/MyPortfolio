Embedded Systems Project

This project involves the use of Microcontrollers (Arduino) to communicate with each other using UART Serial Communication. A PIR Sensor detects motion and sends a signal to the first Arduino to open the servo motor and light the green LED. This Arduino then sends a signal to another Arduino to display "The Door is opened" on an LCD screen.
